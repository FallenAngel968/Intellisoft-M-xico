# INTELLISOFT MÉXICO - Assets

Esta carpeta contiene todos los recursos gráficos del proyecto.

## Estructura de Archivos

### Logos
- `LOGO INTELLISOFT.svg` - Logotipo completo (INTELLISOFT>_) en negro
- `iISOTIPO_MAYORQUE_ 1.svg` - Isotipo únicamente (>_) en negro

### Versiones de Color
Los logos están disponibles en:
- **Versión clara**: Fondo transparente, texto negro, isotipo negro
- **Versión oscura**: Fondo #0a3042, texto blanco, isotipo #36e961

### Uso en Componentes
- **Navbar**: Usa el logo completo en desktop, isotipo + texto en mobile
- **Footer**: Usa el isotipo con texto
- **Hero**: Puede usar cualquier versión según el contexto

### Especificaciones de Color
- Azul corporativo: #0a3042
- Verde isotipo: #36e961
- Texto blanco: #ffffff
- Texto negro: #000000
